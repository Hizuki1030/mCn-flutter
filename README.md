
# mCn-flutter
![](/images/mCn.jpg)

mCnは従来スマートフォンにGroveセンサーを接続できるようにするアダプターです。

## 特徴
- ハードウェア開発なしでスマートフォンに搭載されていないセンサーを拡張できる。
- 対応センサーは100種類以上！

## 対応環境
* Flutter Android環境
